# Интеграция с Telegram (Telegram Integration)

Платформа Poputki.online тесно интегрирована с Telegram в качестве основного интерфейса.

## Виды взаимодействия

### Telegram Mini App
- Основной интерфейс пользователя реализован как WebApp.
- Доступ осуществляется через кнопку в боте или по прямой ссылке.
- Все запросы аутентифицируются данными `initData` от Telegram.

### Telegram Bot (Уведомления)
- Бот используется для отправки транзакционных сообщений:
  - Подтверждение бронирования.
  - Уведомления водителям о новых пассажирах.
  - Личные сообщения о статусе поездок.
- Для рассылок используется `server/utils/telegramBot.js`.

### Телеграм-каналы (Broadcasting)
- При создании новой поездки сервер автоматически делает рассылку в настроенные каналы/группы.
- Список каналов определяется переменной окружения `TELEGRAM_BROADCAST_GROUP_ID`.
- Сообщения в каналах содержат кнопку «Открыть поездку», которая ведет прямо в Mini App.

## Техническая реализация
- Библиотека: `node-telegram-bot-api`.
- Токен бота хранится в переменной `TELEGRAM_BOT_TOKEN`.
- Вебхуки или long polling: В текущей реализации используется long polling (или прямое API взаимодействие).

## Часто используемые функции в коде
- `sendBroadcast(message, rideId)`: Рассылка в публичные каналы.
- `sendPersonalMessage(chatId, text, options)`: Отправка сообщения конкретному пользователю.
