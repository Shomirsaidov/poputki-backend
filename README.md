# Poputki.online - Backend (Серверная часть)

Этот репозиторий содержит серверную часть платформы Poputki.online, реализованную на Node.js и Express.

## Быстрый старт

1. **Установка зависимостей:**
   ```bash
   npm install
   ```

2. **Настройка переменных окружения:**
   Создайте файл `.env` в корневой директории и добавьте следующие переменные:
   ```env
   PORT=5001
   SUPABASE_URL=ваш_url_supabase
   SUPABASE_ANON_KEY=ваш_ключ_supabase
   TELEGRAM_BOT_TOKEN=ваш_токен_бота
   TELEGRAM_BROADCAST_GROUP_ID=ид_канала_для_рассылок
   MINI_APP_URL=https://poputki.online
   ```

3. **Запуск сервера:**
   - В режиме разработки:
     ```bash
     npm run dev
     ```
   - В обычном режиме:
     ```bash
     npm start
     ```

## Основные возможности

- **Аутентификация**: Интеграция с Telegram WebApp данных.
- **Управление поездками**: API для создания, поиска и бронирования поездок.
- **Уведомления**: Автоматическая рассылка в Telegram каналы и личные сообщения.
- **Админ-панель**: Эндпоинты для получения статистики и управления данными.
